Установка MongoDB
 -https://www.mongodb.com/download-center скачать
 -в консосли cd <install directory>\bin\run mongod


Скачивани RestAPI
git clone https://github.com/Tender6795/PhoneBook-Node-epress-react.git


Запуск сервера:
  cd<install directory>\server
  yarn install
  yarn start

Запуск клиента
  cd<install directory>\client
  yarn install
  yarn start
